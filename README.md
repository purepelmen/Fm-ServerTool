# Fm-ServerTool
Facility Manager Server Tool for easy server managment.

# How To
This section helps with the Fm-ServerTool basics.


**Install and prepare game server**
```sh
Fm-ServerTool setup
```

**Run game server**
```sh
Fm-ServerTool run
```

**Update server**
```sh
Fm-ServerTool update
```

**Erase all server files**
```sh
Fm-ServerTool erase
```
